
 Task Manager Application

###  Description

The **Task Manager Application** is a console-based Python program that allows users to manage daily tasks efficiently. Users can add, view, update, and delete tasks using a text-based menu. All tasks are stored in a JSON file so that data persists even after the program is closed.

This project demonstrates core Python concepts such as **Object-Oriented Programming, file handling, modular design, input validation, and error handling**.

---

###  Features

* Add new tasks
* View all tasks
* Update existing tasks
* Delete tasks with confirmation
* Persistent storage using JSON
* Console-based user interface

---

###  Concepts Used

* Object-Oriented Programming (Classes & Objects)
* File Handling (JSON)
* Modular Programming
* Error Handling
* Regular Expressions
* Lists and Conditional Statements

---

###  Technologies

* Python 3
* Standard Python Libraries (`json`, `re`)
